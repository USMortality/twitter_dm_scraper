# Simple Twitter DM Web Scraper via Playwright

# Installation
`npm i`

# Config
```
    set -x TWITTER_USER <your_handle>
    set -x TWITTER_PASSWORD <..> 
    set -x TWITTER_MESSAGE_ID <..> # from: https://x.com/messages/..
    set -x TWITTER_ME_TITLE Me
    set -x TWITTER_THEM_TITLE Them
```

# Run
```
    npx playwright test
```
